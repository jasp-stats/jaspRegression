# The Regression Module

## Overview

<img src='inst/icons/analysis-classical-regression.svg' width='149' height='173' align='right'/>

**JASP Regression module** is a core module for JASP that provides tools for evaluating the association between variables. The Regression module offers a wide range of functionalities, including frequentist and Bayesian correlation, linear regression, logistic regression, and a generalized linear model.


## R Packages

<img src='https://www.r-project.org/logo/Rlogo.svg' width='100' height='78' align='right'/>

The functionality is served by several R packages

- **BAS** — Bayesian variable selection and model averaging for linear models ([BAS on CRAN](https://cran.r-project.org/package=BAS))
- **bstats** — Bayesian statistics utilities ([bstats on GitHub](https://github.com/AlexanderLyNL/bstats))
- **emmeans** — Estimated marginal means and contrasts ([emmeans on CRAN](https://cran.r-project.org/package=emmeans))
- **logistf** — Firth's bias-reduced logistic regression ([logistf on CRAN](https://cran.r-project.org/package=logistf))
- **ppcor** — Partial and semi-partial correlation ([ppcor on CRAN](https://cran.r-project.org/package=ppcor))
- **VGAM** — Vector generalized linear and additive models ([VGAM on CRAN](https://cran.r-project.org/package=VGAM))
